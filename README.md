# 서비스 소개

팝콘은 팝업스토어 정보를 쉽고 편리하게 보실 수 있는 플랫폼입니다.

AI이미지 추천, 좋아요 기반 추천, POPBTI를 통한 추천을 활용하여 당신에게 맞는 팝업스토어를 찾고

AR카메라를 이용한 다양한 인증 사진도 찍으실 수 있습니다.

[팝콘 바로가기](https://i10c211.p.ssafy.io/)
